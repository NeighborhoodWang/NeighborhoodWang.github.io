How to use this application?

This application is very to use.

There are only two input textboxs in order to put your numbers.

There is a radiobutton that you can select your method in order to handle your two numbers.

Ultimately, when you have made your choices you can press the submit button then you will get your result on the right hand panel.

How to get this?

I write this program by R language and a very handy package whose name is shiny.
The key point in my application is a small switch Statement. The application get the method from the user then handle two numbers using specified method.
Here's a example: suppose I choose "Add" method 

notation <- "add"; number1 <- 1; number2 <- 2;
    if(notation == "add")
        result <- as.numeric(number1) + as.numeric(number2)
      if(notation == "sub")
        result <- as.numeric(number1) - as.numeric(number2)
      if(notation == "multi")
        result <- as.numeric(number1) * as.numeric(number2)
      if(notation == "divide")
        result <- as.numeric(number1) / as.numeric(number2)
result

## [1] 3

Then we get the result.
